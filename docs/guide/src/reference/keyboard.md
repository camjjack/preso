# Keyboard Shortcuts

These work in either window while presenting; navigation keeps both windows in
sync.

## Navigation

| Key | Action |
|-----|--------|
| <kbd>→</kbd> <kbd>↓</kbd> <kbd>Space</kbd> <kbd>PageDown</kbd> | Next reveal step, or next slide when fully revealed |
| <kbd>←</kbd> <kbd>↑</kbd> <kbd>Backspace</kbd> <kbd>PageUp</kbd> | Previous step / slide |
| <kbd>Home</kbd> | First slide |
| <kbd>End</kbd> | Last slide |
| *digits* then <kbd>Enter</kbd> | Jump to that slide number |
| <kbd>Esc</kbd> | Toggle the slide overview grid |

In the **overview grid**, click a thumbnail — or type a number and press
<kbd>Enter</kbd> — to jump to that slide. Opening it scrolls to the current
slide (showing the slides before it). While it's open, <kbd>Home</kbd> /
<kbd>End</kbd> jump to the top / bottom and <kbd>PageUp</kbd> /
<kbd>PageDown</kbd> scroll by a page; the arrow keys move the highlighted
slide and the grid follows.

## View

| Key | Action |
|-----|--------|
| <kbd>f</kbd> | Toggle fullscreen for the focused window |
| <kbd>r</kbd> | Reset the talk timer |
| <kbd>v</kbd> | Play the current slide's [video](../writing/video.md): toggle inline play/pause, or launch an external player |

## Annotation

These draw on the live audience window (see [Presenting](../getting-started/presenting.md#annotating-the-slide)):

| Key | Action |
|-----|--------|
| <kbd>l</kbd> | Toggle the laser pointer |
| <kbd>p</kbd> | Toggle pen drawing |
| <kbd>c</kbd> | Clear all annotations |
